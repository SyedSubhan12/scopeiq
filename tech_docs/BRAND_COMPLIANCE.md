# ScopeIQ · Brand Compliance Checklist
**Version:** 1.0 · 2026

Every piece of creative — ad, landing page, email, in-product copy, social post, pitch deck slide, event signage — must answer yes to all five of these questions before it ships.

---

## 1. The Decision Filter
1. **Does this make scope protection feel like expertise, not administration?**  
   *Reframing protection from a chore into a professional capability.*
2. **Would a studio lead show this to a client without embarrassment?**  
   *Ensuring visual and verbal discriminators match professional standards.*
3. **Does this take a side, or does it hedge?**  
   *Structure protects work. Scope creep is a system failure. State what we believe.*
4. **Does this treat the user as a professional, or a victim?**  
   *User is a capable professional working without the right tools, not a victim of bad luck.*
5. **Is there a specific number, artifact, or fact that proves this?**  
   *Authority is earned through economic and technical specificity.*

---

## 2. Voice Comparison Matrix

| Context | Category Default (Warm) | ScopeIQ Voice (Declarative) |
|---|---|---|
| **Empty State** | "No flags yet! Create one to get started 🎉" | "No scope flags. SOW is active. Monitoring engaged." |
| **Hero Title** | "All-in-one platform for creatives" | "You delivered more than you billed. We can prove it." |
| **Error State** | "Oops! Something went wrong. Try again?" | "Parsing failed. Non-standard document format detected." |
| **Onboarding** | "Let's get you set up!" | "Upload your SOW to begin scope monitoring." |
| **CTA** | "Start Free Trial" | "See how it works" |

---

## 3. Visual Register
- **Primary:** Deep Teal (`#0F6E56`) — *Competence without hierarchy.*
- **Typography — Brand:** `Fraunces` or `Instrument Serif` — *Editorial authority.*
- **Typography — Product:** `IBM Plex Sans` or `Inter` — *Technical precision.*
- **Imagery:** Evidence and artifacts (signed COs, flagged cards) NOT lifestyle photography.

---

## 4. Brand Stages
- **Awareness:** Make the financial loss personal.
- **Consideration:** Establish category difference (Platform vs Tool).
- **Activation:** Infrastructure installation mindset.
- **Retention:** Evidence-based value reinforcement.
- **Advocacy:** Provide specific economic language to share.
